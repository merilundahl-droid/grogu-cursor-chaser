# Grogu Spine assets

Export your Grogu from Spine (JSON export) into this folder.

Expected filenames (defaults used by the code):

- `grogu.json`
- `grogu.atlas`
- One or more atlas page images referenced by the atlas, e.g. `grogu.png`

If your files are named differently, either rename them to match, or update the paths in:

- `src/renderer/groguRig.ts`
